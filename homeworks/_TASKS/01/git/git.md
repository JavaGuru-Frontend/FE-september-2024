## команды GIT
- клонировать репозиторий                           -> git clone 			
- обновить локальный репозиторий (на компьютере)    -> git pull  
- создать свою ветку                                -> git create branch 		
- зафиксировать изменения                           -> git commit			
- отправить изменения в репозиторий                 -> git push
